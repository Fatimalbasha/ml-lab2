# ml-lab2 

## Dataset Description
The SMS Spam Collection Dataset contains 5,574 SMS messages written in English.
Each message is labeled as either spam or ham (legitimate).
The dataset is publicly available on Kaggle.

## Machine Learning Problem Type
This task is a supervised machine learning problem.
It is a classification problem where the goal is to classify SMS messages as spam or ham.

## Target Variable
The target variable is the column `v1`, which represents the class label
(spam or ham). 

## Problem Statement
The goal of this project is to build a machine learning model that can automatically classify SMS messages as spam or ham based on their textual content.
